# MI 60001: Introduction to Computer Science and Programming in Python

## Overview
- **Course Name:** MIT 60001
- **Platform:** MIT OpenCourseWare (OCW)
- **Focus:** Computational problem-solving, algorithmic thinking,  and Python programming basics
- **Audience:** Beginners in computer science, or learners transitioning lnto Python

## Structure
- **Lectures:** 12-13 video lectures (1 hour each)
- **Problem Sets:** PS0-PS5 (weekly assignments)
- **Exams:** Midterm + Final (practice versions available)
- **Textbook:** Introduction to Computation and Programming using - - Python* by John Guttag, 2nd Edition

## Learning Goals
- Master Python basics: variables, loops, functions
- Develop algorithmic thinking: break problems into steps
